# 036_Java数据结构与算法_前缀_中缀_后缀表达式规则

## 前缀表达式（波兰表达式）

1. 前缀表达式又称为波兰式，**前缀表达式的运算符位于操作数之前**；
2. **举例说明**：(3+4)x5-6对应的前缀表达式就是 -x+3456；
