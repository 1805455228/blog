# 004_Java数据结构与算法_数据结构与算法的关系

## 数据结构和算法的介绍

### 数据结构和算法的关系

- 数据data结构（structure）是一门研究组织数据方式的学科，有了变成语言也就有了数据结构，学号数据结构可以编写出更加漂亮，更加有效率的代码。
- 要学习好数据结构就要多多考虑如何将生活中遇到的问题，用程序去实现解决。
- 程序=数据结构+算法
- 数据结构是算法的基础，换言之，想要学好算法，需要把数据结构学到位。
